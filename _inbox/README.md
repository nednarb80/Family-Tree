# Inbox — drop new documents here

This is the drop zone for **new records waiting to be added to the tree**: scanned
certificates, census pages, photographed headstones, obituaries, photos.

## How to add one

- **From GitHub (anyone):** click **Add file → Upload files** and drag yours in.
  Give it a descriptive name like `collins-samuel-1900-census.jpg`.
- **On the maintainer's computer:** just save the file into this folder.

## What happens next

The maintainer runs the document-ingest workflow, which:

1. **reads** the document,
2. **matches** the people named to existing pages in the tree,
3. **proposes** exactly what would change (new source, timeline events, confidence
   bump) — nothing is written until it's approved, and
4. **commits** the cited updates and files your image into the right person's
   `documents/` folder.

So anything left here is safe — it won't alter the tree on its own. Once a document
has been processed, it gets moved out of the inbox into the person it documents.

*Tip: original records (a photo of the actual certificate or census sheet) are far
more valuable than a typed summary. Include where it came from if you can.*
